# ระบบทะเบียนเครื่องสำรองไฟ UPS

ระบบจัดการข้อมูลเครื่องสำรองไฟ UPS สำหรับโรงพยาบาลชุมชนหนองแค

## ฟีเจอร์

- 🔐 **ระบบ Login & สมัครสมาชิก** - ตรวจสอบตัวตนและบันทึกผู้ใช้
- 📊 **แดชบอร์ด** - แสดงสถิติและข้อมูลสรุปของเครื่อง UPS
- ➕ **เพิ่มเครื่องใหม่** - ฟอร์มบันทึกข้อมูลเครื่องสำรองไฟใหม่
- 📋 **ตารางข้อมูล** - แสดงรายการเครื่อง UPS ทั้งหมด 137 เครื่อง
- 🔍 **ค้นหาและกรอง** - ค้นหาเครื่องตามหมายเลข รุ่น ที่ตั้ง สถานะ
- ⚠️ **เตือนเครื่องที่ต้องเปลี่ยนแบต** - แสดงรายการเครื่องที่ต้องการดำเนินการ

## การใช้งาน

### หน้า Login
1. ใส่อีเมลและรหัสผ่าน
2. หากไม่มีบัญชี ให้คลิก "สมัครสมาชิก"
3. กรอกข้อมูล: ชื่อ, อีเมล, รหัสผ่าน

### หน้า Dashboard
- แสดงจำนวนเครื่องทั้งหมด
- เครื่องที่ต้องเปลี่ยนแบตเตอรี่
- เครื่องที่เสีย
- เครื่องที่ใช้งานได้

### หน้าเพิ่มเครื่องใหม่
กรอกข้อมูล:
- หมายเลขเครื่อง
- รุ่น
- ห้องที่ติดตั้ง
- คอม/ตู้ที่ติดตั้ง
- วันติดตั้ง
- ประกัน UPS
- อุปกรณ์กันไฟกระชาก
- สถานะ
- การดำเนินการ

### หน้าตารางข้อมูล
- ค้นหาเครื่องด้วยคำค้น
- กรองตามสถานะ (ใช้งานได้/ต้องเปลี่ยนแบต/เสีย)
- ลบข้อมูลเครื่องที่ไม่ต้องการ

## เทคโนโลยี

- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla JS)
- LocalStorage API (เก็บข้อมูล)

## ข้อมูลเครื่อง

- **รวมเครื่องทั้งหมด**: 137 เครื่อง
- **ประเภท**: Zircon, ETECH Thor, Syndome, Cyber Power เป็นต้น
- **ที่ตั้ง**: โรงพยาบาลชุมชนหนองแค

## หมายเหตุ

- ข้อมูลจะถูกบันทึกใน LocalStorage ของเบราว์เซอร์
- สามารถใช้งานแบบ Offline
- เมื่อเคลียร์ข้อมูลของเบราว์เซอร์จะหายไป

## ผู้พัฒนา

สร้างด้วย ❤️ สำหรับการจัดการเครื่องสำรองไฟ UPS
